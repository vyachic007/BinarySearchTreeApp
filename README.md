# Описание проекта: Бинарное поисковое дерево

В данном проекте реализуется работа с бинарным поисковым деревом (BST) на основе входных данных из текстового файла.

## Основные функции приложения:

- Построение в связной памяти бинарного поискового дерева по данным из текстового файла, используя операцию поиска с вставкой.
- Поиск вершин дерева, у которых количество потомков в левом поддереве отличается от количества потомков в правом поддереве ровно на единицу, с использованием рекурсивного восходящего обхода (postorder).
- Нахождение высоты дерева на основе итеративного нисходящего обхода (preorder) с явным использованием стека.
- Поиск k-го листа в порядке слева направо с использованием рекурсивного смешанного обхода (inorder).
- Удаление вершины по заданному ключу с помощью операции поиска и исключения.

## Структура данных:

- `TreeNode` — класс узла дерева, содержит ключ и ссылки на левого и правого потомков, а также методы для проверки типа узла (лист, потомок слева или справа).
- `BalancedNodeInfo` — вспомогательный класс для хранения информации об узлах, где различие количества потомков слева и справа равно единице.
- `BinarySearchTree` — реализует операции вставки, поиска, удаления и проверки пустоты дерева.
- `TreeAnalyzer` — реализует алгоритмы обхода и анализа дерева для поиска сбалансированных узлов, вычисления высоты и поиска k-го листа.
- `FileService` — отвечает за чтение целых чисел из файла для построения дерева.
